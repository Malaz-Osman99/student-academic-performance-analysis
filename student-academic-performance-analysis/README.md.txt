# Student Academic Performance Analysis

This project explores the factors that may influence student academic performance using a dataset containing demographic, social, and lifestyle variables.

The analysis includes data cleaning using Python (Pandas) and exploratory analysis using Excel Pivot Tables and visualizations.

## Objectives

- Investigate how demographic and lifestyle factors relate to academic performance
- Explore relationships between study habits and student grades
- Identify patterns that may affect student achievement

## Dataset Features

The dataset includes:

- Gender
- Race/Ethnicity
- Parental Education
- Lunch Type
- Test Preparation
- Study Time per Week
- Daily Sleep Duration
- School Type
- Internet Access
- Math Score
- Reading Score
- Writing Score

## Tools Used

- Python (Pandas)
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Project Steps

1. Data Cleaning using Python
2. Handling Missing Values
3. Data Type Conversion
4. Feature Engineering (Average Score)
5. Exploratory Data Analysis
6. Pivot Table Analysis
7. Visualization
8. Insight Generation
